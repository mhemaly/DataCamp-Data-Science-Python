# [Statistics Fundamentals with Python Track](https://learn.datacamp.com/skill-tracks/statistics-fundamentals-with-python?version=1)

## Track Description
Statistical knowledge is key to evaluating, interpreting, and reporting findings from your data. With these courses, you will learn to confidently evaluate statistical models, simulate data, and draw conclusions from a wide variety of data sets.

## Track Courses:
1. [Statistical Thinking in Python (Part 1)](https://learn.datacamp.com/courses/statistical-thinking-in-python-part-1)
2. [Statistical Thinking in Python (Part 2)](https://learn.datacamp.com/courses/statistical-thinking-in-python-part-2)
3. [Introduction to Linear Modeling in Python](https://learn.datacamp.com/courses/introduction-to-linear-modeling-in-python)
4. [Statistical Simulation in Python](https://learn.datacamp.com/courses/statistical-simulation-in-python)
5. [Case Studies in Statistical Thinking](https://learn.datacamp.com/courses/case-studies-in-statistical-thinking)
